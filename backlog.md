# MamaBot Backlog

> Дата актуализации: 2026-06-16

## Статус проекта

**Готово:**
- Онбординг (выбор языка, ввод ПМ, подтверждение ПДР)
- Главное меню (6 кнопок, 4 ряда: Моя неделя, Дневник настроения, Питание, Пригласить партнёра, Настройки, Помощь)
- Настройки (смена языка, просмотр ПМ, сброс данных)
- Еженедельная рассылка уведомлений
- Коллекция `pregnancy_data` с данными по 40 неделям (RU/EN)
- Схемы и валидация: `mood_logs`, `nutrition_logs`, `partners`
- Firestore security rules + тесты
- Раздел «Моя неделя» — showWeekInfo, навигация prev/next (14 тестов)
- Раздел «Дневник настроения» — 2-step mood+energy, история, статистика (15 тестов)
- Раздел «Питание» — выбор mealType, текстовый ввод foods, история (12 тестов)
- Раздел «Пригласить партнёра» — генерация кода, ввод, статус (19 тестов)
- Slash-команды: /week, /mood, /nutrition, /invite, /menu, /settings, /help, /start
- Интеграционные тесты: 6 сквозных сценариев (3 проходят, 3 выявили баги)
- Unit-тесты: 583 passed

**Известные баги (выявлены интеграционными тестами):**
- Mood/Nutrition — `createMoodLog`/`createNutritionLog` фабрики документов, данные не сохраняются в Firestore
- Week info — требуется seed `pregnancy_data` в эмуляторе для интеграционных тестов
